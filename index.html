Parent tasks average child progress so your dashboard reflects real effort. A huge branch and a tiny checklist are weighted fairly.

FAQ

Short answers for people comparing Stratum with Trello, Notion, and other project management systems.

Is Stratum a free Trello alternative?

Yes. Stratum is free to use and designed as an open-source Trello alternative for hierarchical project planning and Kanban workflows.

Can I self-host Stratum?

Yes. You can run it on your own Firebase instance or use the free public version, depending on how you want to manage data and deployment.

How is Stratum different from Trello?

Trello is centered on flat boards. Stratum adds infinite nested tasks, progress rollups, next-action pins, and a tree view for deeper planning.

Is Stratum a Notion alternative?

Yes. If you want a lighter project management system than Notion but still need nested tasks and board views, Stratum is designed for that workflow.

Does Stratum support Kanban?

Yes. Kanban is one of the core views, and task status stays synchronized across both the tree and board experiences.
